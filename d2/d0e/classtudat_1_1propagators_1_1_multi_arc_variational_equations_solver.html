inline

Function to return the numerical solution history of integrated variational equations, per arc. Each vector entry contains the results of a single arc, stored in a vector of maps. Inner vector has size two: first entry is state transition matrix history, second is sensitivity matrix history, both stored as maps. Key of map denotes time, values are matrices.

Returns
Numerical solution history of integrated variational equations, per arc.
