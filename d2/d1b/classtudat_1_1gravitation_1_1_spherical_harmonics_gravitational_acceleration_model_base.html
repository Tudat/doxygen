Detailed Description

template<typename StateMatrix>
class tudat::gravitation::SphericalHarmonicsGravitationalAccelerationModelBase< StateMatrix >

This template class serves as the base class for the SphericalHarmonicsGravitationalAccelerationModel, CentralGravitationalAccelerationModel, CentralJ2GravitationalAccelerationModel, CentralJ2J3GravitationalAccelerationModel, and CentralJ2J3J4GravitationalAccelerationModel classes.

Template Parameters
StateMatrixType used to store a state matrix.
