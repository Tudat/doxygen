Detailed Description

Class to compute the second degree gravitational torque: the gravitational torque exerted by a point mass on a body with degree two gravity field (parameterized by inertia tensor).

Updates member variables used by the gravitational accfeleration model. Function pointers to retrieve the current values of quantities from which the torque is to be calculated are set by constructor. This function calls them to update the associated variables to their current state.
