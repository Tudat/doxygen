void tudat::interpolators::OneDimensionalInterpolator< IndependentVariableType, DependentVariableType >::makeLookupScheme ( const AvailableLookupScheme  selectedScheme)
inlineprotected

This function creates the look-up scheme that is to be used in determining the interval of the independent variable grid where the interpolation is to be performed. It takes the type of lookup scheme as an enum and constructs the look-up scheme from the independentValues_ that have been set previously.

Parameters
