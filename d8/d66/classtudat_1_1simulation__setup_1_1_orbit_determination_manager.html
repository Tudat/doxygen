Detailed Description

template<typename ObservationScalarType = double, typename TimeType = double>
class tudat::simulation_setup::OrbitDeterminationManager< ObservationScalarType, TimeType >

Top-level class for performing orbit determination. All required propagation/estimation settings are provided to this class, which then creates all objects needed for the propagation and estimation process. The parameter estimation itself is performed by providing measurement data and related metadata (as PodInput) to the estimateParameters function.
