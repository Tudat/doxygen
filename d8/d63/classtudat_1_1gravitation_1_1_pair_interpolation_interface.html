Detailed Description

Interface class between GravityFieldVariations objects that are interpolated and TimeDependentSphericalHarmonicsGravityField.

All correction coefficients are calculated as rectnagular blocks in both the cosine and sine matrices (of equal size)

Constructor, receives an interpolator (typically created from a GravityFieldVariations object by the createInterpolatedSphericalHarmonicCorrectionFunctions function), which is used to approximate the spherical harmonic coefficient corrections at any given time (inside the interpolation window). All correction coefficients are calculated as rectangular blocks in both the cosine and sine matrices (of equal size)
