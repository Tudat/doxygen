Detailed Description

Class to determine the trimmed angle-of-attack for a given set of aerodynamic coefficients. The coefficient interface provided as input must be dependent on the angle of attack for this class to function.

Function to find the trimmed angle of attack for a given set of independent variables. This function iteratively changes the angle of attack in the untrimmedIndependentVariables vector (keeping any other variables the same) and returns the value of the trimmed angle of attack. Note that this function will typically have some small numerical error in the result, as a result of the error tolerances in the root finder.
