Detailed Description

Interface class for the estimation of a constant relative observation bias (at given link ends and observable type). Unlike most other EstimatableParameter derived classes, this class does not have direct access to the class (ConstantRelativeObservationBias) used in the simulations for the observation bias. This is due to the fact that the ConstantRelativeObservationBias class is templated by the observable size, while this class is not.
