Detailed Description

EphemerisSettings derived class for defining settings of a ephemeris interpolated from Spice data.

Retrieving a state from Spice can be very computationally intensive. Using the settings of this class, ephemeris data for a body is pre-computed using a limited number of calls to Spice, which is then used to create an interpolator (6th order Lagrange). For many numerical integration scenarios, this approach may be faster than using DirectSpiceEphemerisSettings, with negligible influence on accuracy.
